# SkpCrm Python Flask Api

## Instalation

1. Create a virtual environment
Windows
```
cd \skpcrm
\skpcrm> mkvirtualenv skpcrm
```
2. Activate the environemt
Windows
```
\skpcrm> activate
```
3. Install Application dependencies
Windows
```
cd src\app
\skpcrm\src\app> pip install -r requirements.txt
```
